## day5

vim 的可视化模式：  
字符： v
行： V
块： C-v

如何退出可视化模式：  
`esc, C-[`, 再次输入 v 或 V

导航:  
切换可视化光标: o  
回到上一次选择的区域: gv

跨多行编辑:  
给多个行的后面加分号: 配合 A 或者 I  
赋值粘贴: 利用系统的剪切板
===

```
const a = 1 dfdf
const b = 2 dfdf
const c = 3dfdf

const d = 111
const e = 211
const f = 311


const g = 1
const h = 2
const i = 3
```
